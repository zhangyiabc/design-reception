# client-app
**本项目介绍**
本项目是一个多人博客系统 基本的功能已经实现，基于vue2实现
本项目使用nodejs和mysql提供服务
[服务端项目地址](https://github.com/zhangyiabc/design-server)
[后台项目地址](https://github.com/zhangyiabc/server-blog)

## 项目拉下来第一步
```
npm install
```

## 第二步
```
npm run serve
```

## 打包
```
npm run build
```

## Lints and fixes files
```
npm run lint
```
